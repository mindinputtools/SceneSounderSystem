These scripts and service files are used to make SceneSounder work on a newly installed system.
The install_deps.sh script need to be run on every system to run SceneSounder. The install_service.sh should be used when you what SceneSounder to start at boot time, like on a normal user system.
The install_dotnet.sh needs only to be used if this is a system where you want to perform development of the SceneSounder system.
